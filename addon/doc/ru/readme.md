# Доступность Telegram Desktop для NVDA

## Обзор

Telegram Desktop Accessibility - это дополнение NVDA для Telegram Desktop в Windows. Оно помогает NVDA продолжать читать элементы Telegram, когда Telegram предоставляет сведения, которые иначе могут прервать озвучивание фокуса.

Дополнение намеренно остается небольшим. Оно не добавляет новые функции Telegram, не изменяет ваши сообщения и никуда не отправляет данные учетной записи. Его основная задача - сделать затронутые экраны Telegram удобнее для чтения с NVDA.

> [!IMPORTANT]
> Дополнение улучшает только те экраны Telegram Desktop, которые умеет распознавать. Если Telegram изменит интерфейс, часть поведения может измениться до обновления дополнения.

## Возможности

* Улучшает переход фокуса по `Tab` и озвучивание фокуса в списках чатов и сообщений Telegram.
* Помогает NVDA читать список стран или регионов при входе по номеру телефона.
* Сохраняет собственные названия элементов Telegram, поэтому названия чатов, сообщений и стран озвучиваются так, как их предоставляет Telegram.
* Работает в фоне и не требует запоминать дополнительные команды дополнения.

## Советы

* При работе с Telegram через NVDA держите окно Telegram Desktop обычным и видимым.
* Если после обновления Telegram список чатов перестал читаться, сначала перезапустите Telegram Desktop и NVDA.
* Если проблема остается, сообщите версию Telegram Desktop, версию NVDA и экран, на котором пропадает речь.

## Переключение Telegram на русский язык

1. Откройте [ссылку русского языка](https://t.me/setlanguage/ru-beta).
2. Когда Telegram откроет диалог подтверждения смены языка, нажимайте `Tab`, пока не найдете кнопку **Apply Language**, затем активируйте ее.
3. Если часть текста не обновилась сразу, закройте и снова откройте Telegram Desktop.

## Что исправляет дополнение

На некоторых экранах Telegram Desktop 6.8.3 и более поздних версий NVDA может перестать читать строку с текущим фокусом или не прочитать текущую строку, когда `Tab` попадает на контейнер списка. Самые заметные примеры - список чатов и список сообщений: вы перемещаетесь по элементам, но NVDA может молчать.

Дополнение восстанавливает фокусируемость затронутых контейнеров списков Telegram на стороне NVDA и обходит проблемные сведения выбора в затронутых строках, чтобы NVDA продолжала читать текущий элемент. Такая же защита строк используется в диалоге выбора страны или региона для номера телефона.

## Клавиатурные команды

Это дополнение не добавляет собственных клавиатурных команд. Используйте встроенные команды Telegram Desktop и обычные команды навигации NVDA.

## Сообщество и поддержка

* **Официальный канал Telegram**: [tdesktopnvda](https://t.me/tdesktopnvda). Подпишитесь на канал, чтобы получать заметки о выпусках и новости проекта.
* **Группа пользователей Telegram**: [tdesktopnvda_group](https://t.me/tdesktopnvda_group). В группе можно задавать вопросы, делиться отзывами и обсуждать проблемы доступности.
* **Исходный код и отслеживание проблем**: [keyang556/tdesktopnvda](https://github.com/keyang556/tdesktopnvda). Если вы нашли ошибку или хотите предложить функцию, откройте Issue. Pull Request с кодом, документацией или переводом приветствуется.
* **Контакт разработчика**: Ken Chang <lindsay714322@gmail.com>

## Поддерживаемые версии

* Telegram Desktop для Windows 6.8.3 и более поздних версий, особенно версии с той же проблемой фокуса или чтения списков.
* NVDA 2024.1 или более поздней версии.

## Сборка из исходного кода

В корне этого репозитория выполните:

```powershell
uv run scons
```

Созданный пакет `.nvda-addon` можно установить через менеджер дополнений NVDA.
