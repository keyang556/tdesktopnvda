# Доступность Telegram Desktop для NVDA

## Обзор

«Доступность Telegram Desktop» — дополнение NVDA для Telegram Desktop в Windows. Оно добавляет две команды быстрого перехода, не изменяя встроенные функции доступности и названия элементов Telegram.

## Возможности

* `Alt+1` перемещает фокус на выбранный чат в списке чатов, а если чат не выбран — на первый чат.
* `Alt+M` открывает главное меню Telegram.
* Список чатов определяется по стабильным сведениям о классе UIA, а не по переведённому названию элемента, поэтому команды не зависят от языка интерфейса Telegram.
* Названия чатов, сообщений, кнопок и элементов списков по-прежнему предоставляет Telegram. Дополнение не заменяет и не изменяет их.

## Использование

Установите дополнение, перезапустите NVDA по запросу и используйте две команды дополнения в главном окне Telegram. Дополнительная настройка не требуется.

Если `Alt+1` не может найти список чатов или список пуст, NVDA сообщит причину. Если на текущем экране Telegram главное меню недоступно, при нажатии `Alt+M` NVDA сообщит об этом.

## Реализация

Исправленный поставщик специальных возможностей Qt в Telegram Desktop предоставляет через UIA имена классов RTTI. Дополнение определяет список чатов по классу `Dialogs::InnerWidget`, не сравнивая локализованное доступное имя. Команда главного меню находит встроенную кнопку меню Telegram внутри `Dialogs::Widget` и вызывает её стандартное действие.

## Сочетания клавиш

### Дополнение

| Сочетание клавиш | Предоставлено | Действие |
|---|---|---|
| **Alt+1** | Дополнение | Переместить фокус в список чатов |
| **Alt+M** | Дополнение | Открыть главное меню |

### Чаты

| Сочетание клавиш | Предоставлено | Действие |
|---|---|---|
| **Стрелка вверх / вниз / Page Up / Page Down** | Telegram Desktop | Навигация внутри чата |
| **Shift+прокрутка** | Telegram Desktop | Ускоренная навигация внутри чата |

| **Ctrl+Shift+Tab / Ctrl+Page Up / Alt+стрелка вверх** | Telegram Desktop | Перейти к предыдущему чату |

## Поддерживаемые версии

* Telegram Desktop для Windows 7.0.1 или более поздней версии.
* NVDA 2024.1 или более поздней версии.

## Сборка из исходного кода

В корне репозитория выполните:

```powershell
uv run scons
```

Созданный пакет `.nvda-addon` можно установить через магазин дополнений NVDA.
